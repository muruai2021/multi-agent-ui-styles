# 输出模板

> 开发者Agent使用此文档生成标准化输出。

## Tailwind Config 模板

```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{html,js,jsx,ts,tsx,vue,svelte}'],
  theme: {
    extend: {
      colors: {
        // === [STYLE_NAME] Design System ===
        // Background
        'bg-primary': '{PRIMARY_BG}',

        'bg-inverse': '{INVERSE_BG}',

        'text-secondary': '{TEXT_SECONDARY}',
```

## CSS Variables 模板

```css

     [STYLE_NAME] Design System
     Generated by Multi-Agent UI Styles

  /* --- Color System --- */

  --radius-md: {RADIUS_DEFAULT};

  --radius-xl: {RADIUS_XL};
  --radius-2xl: {RADIUS_2XL};
  --radius-full: 9999px;

  /* --- Shadows --- */
  --shadow-sm: {SHADOW_SM};
  --shadow-md: {SHADOW_DEFAULT};

  --shadow-xl: {SHADOW_XL};
```

## 输出文件命名规范

| 输出类型 | 文件名格式 | 示例 |
|----------|-----------|------|
| Tailwind Config | `[style-name].tailwind.config.js` | `linear.tailwind.config.js` |
| CSS Variables | `[style-name].css` | `linear.css` |
| Design Tokens | `[style-name].tokens.json` | `linear.tokens.json` |
| 完整包 | `[style-name]-design-system.zip` | `linear-design-system.zip` |
